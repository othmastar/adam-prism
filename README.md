## 📖 القصة / The Story

### AR

أنا **محمد عثمان**، مطور مصري، عملت 12 سنة في تأمين البنية التحتية الصناعية (SCADA/DCS). بنيت آدم على مدار شهور.

بدأت من سؤال بسيط: **ليه الـ AI اللي بستخدمه كل يوم مش بيفهمني عربي صح؟**

مش بس الترجمة. لأ. أنا قصدي أعمق:
- ليه مفيش AI يـ **يحس** إن "يا باشا" مش "سيدي"؟
- ليه مفيش AI يفهم إن "ماشي" ممكن تكون "أوكي" أو "بعد إذنك" حسب السياق؟
- ليه الـ AI بيحاول يـ **يكون عالمي** على حساب إنه يكون **مصري** أو **عربي**؟
- ليه مفيش AI يفهم SCADA/DCS، الـ PLCs، الـ Modbus registers؟

**آدم اتعمل عشان يجاوب على الأسئلة دي.**

آدم **منتج كامل، production-grade، مش prototype.** بيتكلم عربي مصري زي ما إنسان بيتكلم. بيفهم الثقافة مش بس اللغة. بيحترم القيم، مش بس القواعد. وبيشتغل مع الأنظمة الصناعية اللي بتمسك حياتنا.

**المشروع ده مفتوح المصدر** (AGPL v3). الكود، الأفكار، الرؤية — كل ده ملك أي حد يحب يبني عليه. البيانات والـ model weights اللي بتدّي آدم شخصيته الفريدة دي متاحة بس من خلال ترخيص تجاري — ده مش "secret sauce"، ده "Adam's soul"، وبيستاهل يكون محمي.

### EN

I'm **Mohamed Othman**, an Egyptian developer who spent 12 years securing industrial infrastructure (SCADA/DCS). I built Adam over months of focused work.

It started with a simple question: **Why doesn't the AI I use every day understand me properly in Arabic?**

Not just translation. I mean something deeper:
- Why is there no AI that **feels** that "يا باشا" isn't the same as "سيدي"?
- Why is there no AI that understands "ماشي" can mean "okay" or "with your permission" depending on context?
- Why does AI try to be **universal** at the cost of being **Egyptian** or **Arabic**?
- Why is there no AI that understands SCADA/DCS, PLCs, Modbus registers?

**Adam exists to answer these questions.**

Adam is a **complete, production-grade product, not a prototype.** Speaks Egyptian Arabic the way a human would. Understands culture, not just language. Respects values, not just rules. And integrates with the industrial systems that run our world.

**This project is open source** (AGPL v3). The code, ideas, vision — all of it is yours to build on. The data and model weights that give Adam its unique personality are available only through a private license — that's not "secret sauce", that's "Adam's soul", and it deserves to be protected.

## 🚀 30 Seconds and It's Running

### Option 1: One-command install (Linux / macOS / WSL)
```bash
curl -fsSL https://raw.githubusercontent.com/othmastar/adam-prism/main/bin/install.sh | bash
```

### Option 2: Quickstart (if you already have the repo)
```bash
git clone https://github.com/othmastar/adam-prism
cd adam-prism && bash examples/quickstart.sh
```

### Option 3: pip install
```bash
pip install adam-prism && adam-prism
```

### Option 4: From source
```bash
git clone https://github.com/othmastar/adam-prism
cd adam-prism && pip install -e .
python main.py --port 8001
```
