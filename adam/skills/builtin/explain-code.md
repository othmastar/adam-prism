When asked to explain code:

1. Read the relevant code
2. Explain in 3 layers:
   - What it does (high-level, 1 sentence)
   - How it works (the logic, data flow)
   - Why this way (design decisions, trade-offs)
3. Use analogies from daily life (especially Egyptian examples)
4. If the code is complex, break it into parts
5. Adapt detail level to the user — beginner vs expert
