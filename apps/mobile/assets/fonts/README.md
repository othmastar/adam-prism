# Font Files Required

This directory should contain the Noto Sans Arabic font files. Download them from:
https://fonts.google.com/noto/specimen/Noto+Sans+Arabic

Required files:
- NotoSansArabic-Regular.ttf
- NotoSansArabic-Medium.ttf
- NotoSansArabic-SemiBold.ttf
- NotoSansArabic-Bold.ttf

If these fonts are not available, the app will fall back to the default system font.
