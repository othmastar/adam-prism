## ازاي تستخدم النظام

```bash
# 1. تكلم الموديل مباشرة في terminal
ollama run gemma4:e4b

# 2. تكلم آدم (المدمج)
ollama run adam-v8

# 3. آدم بريزم API
curl http://localhost:8000/api/chat \
  -d '{"message":"ازيك يا آدم"}' \
  -H "Content-Type: application/json"

# 4. LoRA server (مصري مضمون)
curl http://localhost:8080/v1/chat/completions \
  -d '{"messages":[{"role":"user","content":"ازيك يا آدم"}]}'
```
